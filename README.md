# Mactory

**맥토리**는 [`Minecraft`](https://en.wikipedia.org/wiki/Minecraft)내에서 제작된 공장 게임으로,  
게임 내에서 자원을 추출하고, 단순한 자원들로 복잡한 아이템들을 제작하며, 무수히 많은 공장을 건설하여 즐기면 되는 아주 간단한 형식의 **1인용** 게임입니다.

시스템을 위해 전용 리소스팩이 쓰이기도 합니다.
